# 发布相机节点
    ros2 launch depthai_examples rgb_stereo_node.launch.py

# 主要话题与发布节点
    ros2 topic info /color/video/image -v
    ros2 topic info /stereo/depth -v

    /color/video/image   （彩色图像，sensor_msgs/Image）
    /stereo/depth        （深度图像，sensor_msgs/Image）

    通过 ros2 topic info -v 可确认发布者节点：
    - /stereo/depth 的发布节点：rgb_stereo_node

# 使用 image_tools 查看图像
    可以先用 ros2 topic list 找到相机话题：
    ros2 topic list | grep -E '/image$|/depth$'

    查看彩色图：
    ros2 run image_tools showimage --ros-args -r image:=/color/video/image

    查看深度图：
    并不可行，会出现编码问题
    ros2 run image_tools showimage --ros-args -r image:=/stereo/depth

# 查看图像建议使用 rviz2
    出现问题：不能使用ubuntu商店自带的vscode（即snap版本）终端启动rviz2。
    使用官网下载的deb版本最好
